KIến trúc gợi ý
Location + Geofencing : theo dõi GPS và tạo “ điểm quan tâm ” (POI ) với bán kính (geofence)
Geofence engine : quy định khi nào phát thuyết minh (vào vùng/ đến gần)
Narration Engine: chọn text to speeech hoặc là phát audio có sẵn, quản lý hàng đợi, chống trùng lặp
Content layer : Dữ liệu POI offline (SQLite) hoặc là đồng bộ dữ liệu từ server khi có mạng
UI/UX : Bản đồ (Map), danh sách POI, cài đặt độ nhạy GPS/ bán kính, chọn giọng Text to speech, tải gói offline
Framework .Net MAUI (Android/IOS)
GPS và background:
+ Android : FusedLocation Provider Client (thông qua dependency service hoặc là EssentialsGeoLocation), Foreground Service để tạo tracking nền.
+IOS : CLLocationManager với quyền always region monitoring (geofence)
Geofencing: Native API hoặc là tự tính khoảng cách bằng Haversine rồi trigger theo ngưỡng
Text to speech/Audio:
+Text to speech bản địa (Android Text ti speech, IOS AVSpeech Syntheszer) hoặc Azura Cognitive Services (Nếu cần giọng tự nhiên lưu ý vấn đề offline và online)
Map: Microsoft MAUI Controls Maps (MAUI MAPS) hoặc là GGMAP SDK/ Map KIT Binding nếu cần tính năng nâng cao
Offfline: SQLite (EF CORE/SQLite-net) + file âm thanh tải trước. Bản đồ offline có thể cần SDK bên thứ 3 (ví dụ Mapbox, Here) nếu muốn cache/offline mapping thật sự
